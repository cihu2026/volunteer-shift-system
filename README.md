# 志工／老師選班系統

這是一個 GitHub Pages 前端示範版，用來讓志工或老師查詢自己的班表、選班、確認收到提醒，並讓管理員檢查誰尚未確認。

## 已建立檔案

- `index.html`：首頁與畫面結構
- `style.css`：頁面樣式
- `app.js`：選班、查詢、確認與管理員示範邏輯

## 測試資料

可以輸入以下姓名或代碼測試：

- `T001` / 王老師
- `T002` / 陳老師
- `T003` / 林老師

## 目前功能

- 輸入姓名或代碼查詢個人班表
- 選擇示範班別
- 檢查班別名額是否額滿
- 按「我知道了」確認收到提醒
- 管理員檢查已選班與未確認名單

## 重要限制

這一版是純前端示範，資料存在瀏覽器 `localStorage`，不會同步到其他人的手機或管理員後台。

正式給 200 位老師使用前，建議下一版接：

1. Google Sheets + Apps Script：最快做自動寄信提醒。
2. Firebase：適合多人即時選班、名額鎖定與後台同步。
