志工／老師選班安全版

真實名單與班表留在 Google Sheets；GitHub Pages 只放前端，不公開志工姓名清單。

安全版重點

老師只輸入自己的個人代碼查詢與選班；真實資料由 Google Apps Script 從 Google Sheets 讀取，不寫進公開 GitHub。

零、連接 Google Sheets 後端

把 Apps Script 部署後取得的 Web App URL 貼在這裡。這個網址只存在本機瀏覽器，不會寫進 GitHub。

一、老師登入／查詢

請輸入個人代碼，例如志工學號。安全版不建議用姓名查詢，避免別人猜名字看到班表。

二、可選班別

查詢成功後才會顯示可選班別。選班時會由 Google Sheets 後端檢查名額，避免額滿後還被重複選走。

三、管理員檢查區

輸入 Apps Script 裡設定的 ADMIN_TOKEN，才可以查看全部選班與未確認名單。

安全版：公開 GitHub 只放程式；真實資料保留在 Google Sheets。
